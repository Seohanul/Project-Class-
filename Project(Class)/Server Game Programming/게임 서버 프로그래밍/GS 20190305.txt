서버란

	서버는 네트워크로 연결된 다른 프로그램에게 서비스를 제공하는 프로그램이다.



게임 서버의 종류
	인증 서버
	패치/업데이트 서버
	로비(배틀넷) 서버
	MMO 게임 서버 ** 이거만 할거
		NPC 서버
		DB Cache 서버
		Chat 서버
		Log 서버		
		...








이 수업에서는

	MMO (Massively Multiplayer Online) 게임 서버

		하나의 프로세스에서 몇천명 이상의 동시접속 유저를 서비스하는 프로그램
			ex) 아이온, 테라 등






서버간의 비교

		MMO서버			웹서버			DB서버

클라 간섭	매우심함		거의없다		약하다

딜레이 민감성	매우 민감		적다			다양하다

확장성		적다			매우 크다		보통





	확장성 - cpu 더 많고 메모리 늘리고해서 동접자 수가 늘어나지 않음

		프로그램을 뜯어 고쳐야하는데 어려움








모바일 게임 서버

	기존에는 웹서버 기반
		클릭할때마다 서버 연결
		모든 컨텐츠는 DB에 저장


	현재 트렌드
		실시간 PvP 컨텐츠 증가 : 레이드, 길드전, 공성전
		MMO 서버가 됨 : 리니지 2 Revolution, 리니지 M









게임서버 제작시 고려점

	성능 
		동접자 수와 HotSpot 허용량으로 평가

		개선을 위한 고려사항
			- 알고리즘 개선
			- 오버헤드 감소
			- Multi-core 활용 ****
				MMO는 Multi-core + Multi Cpu 까지 해야함


	

	네트워크 부하
		- 네트워크 연결 개수
		- 통신량 
		- 효율적인 프로토콜


	
	확장성
		성능을 쉽게 늘릴 수 있는가?   - 넥슨도 실패함 (듀랑고)
		정적/동적 분할			- 여러개 컴퓨터에서 나눠서 서비스, 컨텐츠 별로 나눠서 서비스
		연결성	(심리스니스)		


	딜레이
		네트워크 딜레이 은폐 (패킷 제한)


	보안
		클라이언트를 믿지 않는다.   (클라이언트에서 중요한 결정을 하지 않는다)	- 모든 클라는 해킹당했다는 가정하에 서버 만들어야함
		해킹해도 이익이 없도록 설계

		암호화
			해킹을 어렵게 만들어야함







왜 만들기 어렵고 유지관리가 어려운 게임서버를 운영하는가??

	- 보안 때문
	- 서버 컴퓨터에 해킹은 거의 불가능
